# 📅 Kalender Hijriyah + Pasaran Jawa (JavaScript)

Repository ini berisi implementasi kalender Hijriyah menggunakan JavaScript yang dilengkapi dengan pasaran Jawa (Legi, Pahing, Pon, Wage, Kliwon).

## ✨ Fitur
- Konversi tanggal Masehi ke Hijriyah  
- Menampilkan hari pasaran Jawa  
- Ringan dan mudah digunakan  
- Mudah diintegrasikan ke project web apa pun

  tanggalMasehi: "2026-05-02",
